Movie API | Documentation

Objective:

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Features:

  • Returns a list of ALL movies to the user
  • Returns data (description, genre, director, image URL) about a specific movie by title to the user
  • Returns data about a genre by name
  • Returns data about a specific director (bio, birth year, death year) by name
  • Allows new users to register a user account
  • Allows users to update their user account information
  • Allows users to add a movie to their list of favorite movies
  • Allows users to remove a movie from their list of favorite movies
  • Allows existing users to deregister their user account
