Movie API | Documentation

Objective:

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints:

/movies - Returns a list of all movies via GET method

/movies/[title] - Returns data to the user about a specific movie by title via GET method

/genres/[genre] - Returns data about a genre via GET method

/directors/[director_name] - Returns data (bio, birth year, death year) about a specific director via GET method

/users - Allows new user to register via POST method

/users/[user] - Allows user to update user information (username, email, password)

/users/[user]/favorites - Allows users to add a movie to their favorites (showing only text that a movie has been added) via POST method

/users/[user]/favorites - Allows users to remove a movie from their favorites (showing only text that a movie has been removed) via DELETE method

/users - Allows existing users to deregister (showing only a text that user account has been removed) via DELETE method
